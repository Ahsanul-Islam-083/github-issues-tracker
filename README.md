- 1️⃣ What is the difference between var, let, and const?

- >In JS var is function scoped and hoisted with a initial value of undefined so it can be accessed without and error before even declaring it and it can also be redeclare. On the other Hand let and const are block scoped and if we try to access it before declaration causes temporal dead zone error. but the difference is let can be redeclared but the const can't. however the elements of objects and arrays which are declared with const can be modified.

- 2️⃣ What is the spread operator (...)?

- >spread operator ... is used to spread out elements of an array or object.it makes easy to copy or merge elements .

- 3️⃣ What is the difference between map(), filter(), and forEach()?

- >map() returns a new array by changing each element of the original array. filter() also return a new array with the elements which matches the given condition. forEach() doesn't return anything but you can perform actions to each elements of the array.

- 4️⃣ What is an arrow function?

- >arrow function is a shorter syntax then the regular function and unlike regular function it doesn't has its own 'this' and 'arguments'. but it is ideal for short one-liner simple functions then it doesn't need the return keyword.

- 5️⃣ What are template literals?

- >template literals is a way in js to create a string and and it can dynamically embed  variable inside string like, i am ${name}.
